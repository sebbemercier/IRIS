# Recommended Datasets for IRIS (Analytics)

- **Online Retail Dataset (UCI)**: Transactions for a UK-based non-store retail. [UCI ML Repo](https://archive.ics.uci.edu/ml/datasets/Online+Retail)
- **REES46 E-commerce Behavior**: Multi-category user behavior data. [Kaggle](https://www.kaggle.com/mkechinov/ecommerce-behavior-data-from-multi-category-store)
- **Retailrocket Recommender System**: Transactional and behavioral data. [Kaggle](https://www.kaggle.com/retailrocket/ecommerce-dataset)
